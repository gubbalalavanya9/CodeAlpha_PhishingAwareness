# Phishing & Social Engineering Awareness Training

## Overview
This project is a phishing awareness training presentation developed as **Task 2** of the **CodeAlpha Cyber Security Internship**.  
It focuses on educating users about phishing attacks, social engineering techniques, real-world examples, and best practices to stay safe online.
## Objective
- Create awareness about phishing and social engineering attacks  
- Help users identify phishing emails and fake websites  
- Explain common attacker manipulation techniques  
- Promote best practices for phishing prevention  
## Key Topics Covered
- What is phishing and why it is dangerous  
- Anatomy of phishing emails  
- Identifying fake and malicious websites  
- Social engineering tactics used by attackers  
- Real-world phishing examples  
- Best practices to avoid phishing attacks  
- Incident response steps  
- Interactive quiz for engagement  
## Tools Used
- Canva (for presentation design)
## Project Deliverable
- Phishing Awareness Training Presentation (PPT)
## Learning Outcome
This task helped in understanding how phishing attacks exploit human trust and how user awareness plays a critical role in preventing cyber threats.
## Conclusion
Phishing remains one of the most common cyber threats. Awareness, verification, and cautious behavior are the strongest defenses against such attacks.
